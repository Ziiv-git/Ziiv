Perrin Freres Monthly Champagne Sales

  • Build the model based on ARIMA & SARIMA algorithms to forecast monthly sales of Champagne, as due to seasonality captured in the dataset SARIMA performed better.
  • Analysed the dataset automatically using pandas-profiling library then manually preprocessed the data further accordingly.
  • Sorted data based on time, conducted statistical tests like Augmented Dickey Fuller test to check for stationarity, found out values of p(auto-regressive), d(differencing), q(moving average) values using partial correlation plot and auto correlation plots. To remove the statistical, applied automatic differencing method.
