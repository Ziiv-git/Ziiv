Sales Analytics using Alteryx

Hands on alteryx tool using examples rather doing short projects based on real data. Learning process involves analysis tools, machine learning and time forecasting tools.

Analysis Challenge:

  • To derive insights from furniture sales data to help management in taking profitable business decisions.

Tasks done:

  • Connecting data to alteryx tool from tabular and non-tabular data, SQL table, data from cloud storage and finally merging data streams.
  • Data cleansing and improving data quality.
  • Sampling data, preparing data and outputting the cleaned data.
  • Merging tables to create a data mart.
  • Performing analytics/transformation on data mart.
  • Creating a report in alteryx which can be found in the repo.
  • Scheduling workflow.

Machine learning Challenge:

  1. Regression * Predicting house prices
  2. CLassification * Churn prediction * Bank defaults
  3. Model Comparisons

Time Series learning challenge:

  • Forecasting air travel passengers using ARIMA and Exponential Smoothing tools.
